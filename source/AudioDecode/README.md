# 从零开始学习音视频编程技术（42） AAC数据解析  
ffmpeg4.1解码AAC，并使用SDL/RtAuido播放，同时显示音频波形图的例子

这是Qt的工程，建议使用Qt Creator 打开

Qt5.6.2(vs2013/mingw) + ffmpeg4.1  
1.程序可以在linux下直接编译使用。  
2.已经编译好的库为centos6.5(64位)(gcc 版本 4.4.7)下编译的，如需32位的库请自行编译。
